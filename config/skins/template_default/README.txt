La forma recomendad de editar una skin es instalar el plugin Firebug para
Firefox y usar la funcionalidad de Inspector para encontrar los selectores CSS
adecuados.

	http://getfirebug.com/


Para cambiar imágenes basta con encontrar su selector y poner la ruta relativa
a la imagen. Por ejemplo:

	#logo2 img { background: url(imagen_pirulina.png); }


"imagen_pirulina.png" debe estar en el directorio raíz de la skin.

Una vez terminada de editar la skin crear un zip y actualizar la skin en gamersmafia.
Los archivos "style.css" y "config.yml" deben estar siempre en el directorio
raíz de la skin. Si quieres crear carpetas para organizar imágenes, iconos,
etc, es posible hacerlo.


Para bugs y sugerencias:

	http://gamersmafia.com/foros/forum/871


Las skins son privadas por defecto, si quieres que tu skin sea pública avisa
a slnc una vez la tengas terminada.
